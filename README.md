FellowApp, kullanıcıların çalışma oturumlarını yönetmelerine yardımcı olan bir masaüstü uygulamasıdır. Proje JavaFX kullanılarak geliştirilmiş olup Maven ile yapılandırılmıştır.

Gereksinimler:
- JDK 23
- Maven

Nasıl çalıştırılır?

Windows için PowerShell komutları:
cd ...\Bil481Proje\fellowapp
mvn clean
mvn javafx:run
